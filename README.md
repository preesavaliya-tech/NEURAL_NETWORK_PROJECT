# 🧠 Neural Network on MAGIC Gamma Telescope Dataset

## 📌 Project Overview

This project focuses on training, testing, and validating a Neural Network model using the MAGIC Gamma Telescope dataset to classify high-energy particle events as gamma or hadron.

---

## 🚀 Features

* Data preprocessing and normalization
* Neural Network model implementation
* Model training and evaluation
* Performance metrics (accuracy, confusion matrix, etc.)

---

## 📂 Dataset

* MAGIC Gamma Telescope Dataset
* Contains features of particle events detected by telescopes
* Used for binary classification (gamma vs hadron)

---

## 🛠️ Tech Stack

* Python 🐍
* NumPy
* Pandas
* Scikit-learn
* TensorFlow / Keras (if used)
